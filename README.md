## BMI Calculator (IMC)

A modern and responsive Body Mass Index (BMI) calculator. This project was built to practice fundamental web development concepts such as DOM manipulation, CSS styling with gradients, and JavaScript conditional logic.

## Features

* **Precise Calculation**: Computes the BMI based on the weight (kg) and height (m) provided.
* **Real-time Classification**: Automatically categorizes the result (Underweight, Normal, Overweight, Obese).
* **Dynamic Visual Feedback**: The result text color changes (Green, Yellow, or Red) based on the health status.
* **Responsive UI**: Sleek dark theme with a crimson gradient background, fully centered using Flexbox.

##  Technologies Used

* **HTML5**: Semantic structure.
* **CSS3**: Custom styling, including linear gradients and box-shadows.
* **JavaScript (ES6+)**: Logic for calculations and dynamic DOM updates.

## How to Use

1.  Enter your **Weight** in kg.
2.  Enter your **Height** (e.g., 1.75).
3.  Click **Calculate** to see your result and health classification.



## Calculadora de IMC
Uma calculadora de Índice de Massa Corporal (IMC) moderna e responsiva. Este projeto foi desenvolvido para praticar conceitos fundamentais de desenvolvimento web, como manipulação do DOM, estilização com CSS (gradientes) e lógica de programação com condicionais em JavaScript.

## Funcionalidades
Cálculo Preciso: Calcula o IMC com base no peso (kg) e altura (m) informados.

Classificação em Tempo Real: Categoriza automaticamente o resultado (Abaixo do peso, Peso ideal, Sobrepeso, Obesidade).

Feedback Visual Dinâmico: A cor do texto do resultado muda (Verde, Amarelo ou Vermelho) de acordo com o status de saúde para melhorar a experiência do usuário.

Interface Responsiva: Design moderno com tema escuro e fundo em gradiente crimson, totalmente centralizado usando Flexbox.

## Tecnologias Utilizadas
HTML5: Estrutura semântica para a interface.

CSS3: Estilização personalizada, incluindo linear-gradient, box-shadow e Flexbox.

JavaScript (ES6+): Lógica para cálculos, estruturas condicionais e atualizações dinâmicas no HTML.

## Como Usar
Insira seu Peso em kg.

Insira sua Altura (ex: 1.75). Importante: Use ponto em vez de vírgula.

Clique em Calcular para ver seu resultado e a classificação de saúde.
